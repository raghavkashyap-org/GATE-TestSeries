Q17 MCQ Not checked

Not attempted

Which of the following is true about implementing queue using  linked list ?

Correct answer: C
Solution

(c)
To keep the first in first out order a queue can be implemented using linked list in 2 ways
• If a new node is inserted at the beginning of linked list, then in dequeue operation node must
be removed from end.
• If new node is inserted at the end of linked list then, in dequeue operation, node must be
removed from beginning.
