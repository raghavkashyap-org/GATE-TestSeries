Q5 MCQ Not checked
Technical • COMPUTER SCIENCE & IT • +2 / -0.66
Not attempted

Consider the following statements:
S1 : DNS can use either of TCP or UDP, for its query and response messages.
S2 : SMTP is not used for transferring e-mail messages from the recipients’s mail server to the recipients’s personal computer.
S3 : HTTP can use multiple TCP connections between the same client and the server.
S4 : POP protocol is used to download an E-mail from mailbox server to a mail client.
Which of the above statement are correct?

Correct answer: D
Solution

(d)
• TCP involves a connection establishment phase while UDP does not. Using TCP for DNS may
end up involving several TCP connections to be established since several name server may
have to be contacted to translate a name into an IP address. This imposes a high overhead in
delay. Hence DNS uses UDP for its query and response messages.
• SMTP is a push protocol, whereas the given task is pull operation.
• S3 and S4 statements are correct.
