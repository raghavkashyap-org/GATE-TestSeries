How many order pairs of integer (a, b) are needed to guarantee that there are two ordered pairs
(a1, b1) and (a2, b2) such that a1 mod 6 = a2 mod 6 and b1 mod 8 = b2 mod 8?

Correct answer: B
Solution

(b)
For a in (a, b) there different congruence class possible in mod 6 i.e. 0, 1, 2, 3, 4, 5 and 8 different congruence classes possible for mod 8 i.e., 0, 1, 2, 3, 4, 5, 6, 7, 8.
So, number of different order pair where (a1, b1) and (a2, b2) such that a1 mod 6 = a2 mod 6 and b1 mod 8 = b2 mod 8 not possible are = 6 × 8 = 48.
So, to get two pair with given condition we need 48 + 1 = 49.

Q16 MCQ Not checked
Technical • COMPUTER SCIENCE • +2 / -0.66
Not attempted

Consider the following statements regarding relations:
S1 : Every asymmetric relation is irreflexive.
S2 : Every asymmetric relation is antisymmetric.
S3 : There are some relations which are neither reflexive nor irreflexive.
Which of the above statements are true?

Correct answer: A
Solution

(a)
For S1 and S2: If a relation is asymmetric, then it has to be both irreflexive and antisymmetric.
Therefore S1 and S2 are correct.
For S3: It is true that there are some relations which are neither irreflexive (contain at least one self loop) nor reflexive (don’t contain all self loops).
So all the statements are correct.
