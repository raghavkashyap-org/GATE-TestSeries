Q14 MCQ Not checked
Technical • COMPUTER SCIENCE & IT • +1 / -0.33
Not attempted

Let A = {1, 2, 3, 4}. Define a relation R on A as:
(a, b) ∈ R if and only if a mod b = 0
Which of the following is true about relation R?

Correct answer: B
Solution

(b)
For any a ∈ A, a mod a = 0 (a, a) ∈ R.
Symmetric:
(2, 1) ∉ R, because 2 mod 1 = 0 → so (2, 1) ∈ R.
But (1, 2) ∉ R, because 1 mod 2 = 1 ≠ 0.
So not symmetric.
Transitive:
If a mod b = 0 and b mod c = 0, then a mod c = 0.
