Q12 MCQ Not checked
Technical • COMPUTER SCIENCE • +2 / -0.66
Not attempted

Consider the following statements:
S1 : Static allocation can not support recursive function.
S2 : Stack allocation can support pointers but can not deallocate storage at run-time.
S3 : Heap allocation can support pointers and it can allocate or deallocate storage at run-time.
Which of the above statements are true?

Correct answer: C
Solution

(c)
S1 : False: Since static allocation is done for all objects at compile time in case of recursion, it’s not possible for compiler to decide as recursion depends on recursion parameter. So, recursive function can’t be implemented with static storage allocation.
S2 : Stack allocation can support pointers and with help of pointers it can allocate and deallocate dynamic variables and hence can manage run time storage.
S3 : Heap allocation can also manage run time storage and dynamic memory allocation with help of pointers.
