{AB → C, C → D, AF → B, B → E}
Which of the following is/are true?

Correct answer: A, C
Solution

(a, c)
AF is the only candidate key.
• No partial dependency exist, so relation is in 2NF.
• There exist transitive dependency from AF → B → E hence relation is not in 3NF.
• Replacing FD : {C → D} with {AB → D} do not make relation in BCNF.

Q9 MSQ Not checked
Technical • COMPUTER SCIENCE & IT • +2 / -0
Not attempted

In the context of index file structures used in database, consider the following three alternatives for storing data entries in an index.
A1 : Each data entry with key k is an actual data record with search key k.
A2 : Each data entry is a pair (k, rid), where rid is the record id of data record with search key k.
A3 : Each data entry is a pair (k, rid-list), where rid-list is the list of all record id’s with search key k.
Which of the following statements are correct?

Correct answer: A, B, D
Solution

(a, b, d)
(a) If the index directly stores full data record (A1) there’s number need to go elsewhere for the actual data, so it behave like a file itself, not pointer based index.
(b) A2 and A3 have entries point to record. So it does not matter how or where the actual record one stored.
(c) A3 size depends on rid-list size, depends on how many records share the same key.
(d) Instead of repeating (k, rid) for every record (A2) A3 groups all rid together saving space.
