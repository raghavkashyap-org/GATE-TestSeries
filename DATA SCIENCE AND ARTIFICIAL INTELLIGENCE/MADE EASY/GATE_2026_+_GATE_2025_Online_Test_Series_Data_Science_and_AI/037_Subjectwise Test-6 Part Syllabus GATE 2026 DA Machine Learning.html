Consider the following statements:
S1 : Data fragmentation increases as the depth of the decision tree increases.
S2 : Excessive data fragmentation can lead to overfitting in decision trees.
Which of the following is true?

Correct answer: C
Solution

(c)
• In a decision tree, each internal node splits data based on a threshold. As the tree grows
deeper, data is split into smaller subsets at each level. This is called data fragmentation -
fewer data points per node. Deeper trees mean more fragmentation. So, S1 is correct.
• When data becomes highly fragmented, nodes contain very few samples. The tree may then
fit noise instead of patterns, leading to overfitting and poor generalization. So S2 is correct.

Consider the following statements:
S1 : LDA assumes that each class has a Gaussian distribution with a shared covariance matrix.
S2 : LDA always results in nonlinear decision boundaries between the classes.
S3 : Naive Bayes classification assumes that the features are independent within each class.
Which of the following is true?

Correct answer: B
Solution

(b)
• Linear Discriminant Analysis (LDA) is based on the assumption that:
(i) Each class follows a multivariate Gaussian distribution, and
(ii) All classes share the same covariance matrix (homoscedasticity). So, S1 is correct.
• This assumption allows LDA to derive linear decision boundaries analytically.
• LDA produces linear decision boundaries in the input feature space.
• These boundaries are based on solving a linear equation involving the inverse of the shared
covariance matrix and class means.
• Non-linear boundaries may occur in Kernel LDA, but not in standard LDA. So, S2 is incorrect.
• This is the central assumption of Naive Bayes: given the class label, the features are assumed
to be conditionally independent.
• This assumption simplifies the joint likelihood into a product of individual feature likelihoods.
So, S3 is correct.

the within-class scatter matrix becomes singular (non-invertible).
• This causes numerical instability and can degrade LDA's performance unless modifications or
regularization techniques are applied. So, option (d) is correct.
