Which of the following statements is/are correct a heuristic function that makes it ‘consistent’ (or ‘monotonic’)?
S1 : The heuristic function h(n) must be non-negative for all nodes n.
S2 : For any node n and any successor n′ reached from n, the heuristic function must satisfy h(n) ≤ c(n, n′) + h(n′), where c(n, n′) is the step cost from n to n′.

Correct answer: C
Solution

(c)
• Stating that the heuristic function must be non-negative for all nodes, is not a requirement for monotonicity. So S1 is incorrect.
• A heuristic is monotonic if, for any node n and its successor n′, the estimated cost h(n) to the goal is less than or equal to the actual cost to move from n to n′ plus the estimated cost h(n′) from n′ to the goal. So, S2 is correct.
