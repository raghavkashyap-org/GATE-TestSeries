Consider the following statements:
S1 : Increasing the number of independent features always increases the R-squared value of the model.
S2 : Increasing the number of independent features can decrease the adjusted R-squared if the added features do not improve the  odel significantly.
Which of the following is true?

Correct answer: C
Solution

(c)
• Adding more independent variables never decreases the R-squared because it measures how well the model fits the data, and more features can only improve or keep the fit same. So, S1 is correct.
• The Adjusted R-squared penalizes adding unnecessary variables. If the new features don't
add significant explanatory power, adjusted R-squared can decrease. So, S2 is correct.

Q12 MCQ Not checked
Technical • Data Science & Artificial Intelligence • +1 / -0.33
Not attempted

Which of the following metrics is most sensitive to class imbalance and may give a misleading impression of performance?

Correct answer: D
Solution

(d)
In imbalanced classification problems, accuracy can be misleading because a model predicting
only the majority class can still achieve high accuracy. F1-score, which balances precision and
recall, provides a better measure of performance by focusing on how well the model detects the
minority class. Mean Squared Error and R-squared are regression metrics and not suitable for
classification tasks.
