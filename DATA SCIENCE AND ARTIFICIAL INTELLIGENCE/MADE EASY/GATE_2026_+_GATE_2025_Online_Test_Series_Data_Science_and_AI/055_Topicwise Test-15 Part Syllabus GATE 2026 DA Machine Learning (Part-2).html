Consider the following statements:
S1 : SVM can handle both linear and non-linear classification.
S2 : Kernel functions map data to higher dimensions for linear separation.
Which following is correct?

Correct answer: C
Solution

(c)
• SVM is a strong classifier that handles linear data by finding a separating hyperplane. For non-linear data, it uses Kernel functions to work with data that cannot be separated in the original space. Thus, SVM manages both linear and nonlinear problems. So, S1 is correct.
• Kernel functions map data implicitly into a higher-dimensional space, making non-linearly separable data separable by a hyperplane. This Kernel trick enables SVM to classify nonlinear data effectively. So, S2 is correct.

Consider the following statements:
S1 : The bias-variance tradeoff refers to the balance between underfitting and overfitting in predictive models.
S2 : High variance models tend to fit the training data very well but perform poorly on unseen data.
Which following is correct?

Correct answer: C
Solution

(c)
• The bias-variance tradeoff refers to balancing underfitting and overfitting in a model. High bias causes underfitting where the model is too simple to capture the data pattern. High variance causes overfitting where the model fits training data too closely but fails on new data. So, S1 is correct.
• High variance models fit the training data very well, resulting in low training error. However, they perform poorly on unseen data because they do not generalize well, leading to overfitting. So, S2 is correct.

Consider the following statements about Decision Trees:
S1 : A fully grown decision tree has low bias and high variance.
S2 : Pruning a decision tree is a technique to reduce variance at the cost of increasing bias.
Which of the above statements is/are correct?

Correct answer: C
Solution

(c)
• A fully grown tree perfectly fits training data (overfitting), so low bias, but it’s sensitive to small changes → high variance. So, S1 is correct.
• Pruning simplifies the tree → less overfitting (reduces variance), but may not capture all patterns (increases bias). So, S2 is correct.
