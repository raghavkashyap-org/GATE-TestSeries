Correct answer: B
Solution

Consider the following statements:
S1 : A multilayer perceptron uses a nonlinear activation function.
S2 : A single-layer perceptron uses a threshold function.
Which of the following is/are true?

Correct answer: C
Solution

(c)
• Multilayer perceptrons (MLPs) must use nonlinear activation functions (like ReLU, sigmoid, or tanh) in their hidden layers. Without non-linearity, the entire network would collapse into a linear function, regardless of the number of layers. So, S1 is incorrect.
• The original single-layer perceptron (as introduced by Rosenblatt) uses a threshold activation function, often implemented as a step function (outputting 1 if weighted sum ≥ threshold, otherwise 0). This allows it to solve linearly separable problems. So, S2 is correct.

Not attempted

Which of the following statements is true regarding the elbow method in K-means clustering?

Correct answer: C
Solution

(c)
When the number of clusters K increases in K-means clustering, each cluster contains fewer points,
making the points closer to their respective centroids. As a result, the Sum of Squared Errors
(SSE) decreases because the overall distance between points and centroids reduces. Therefore,
SSE never increases with higher values of K.
