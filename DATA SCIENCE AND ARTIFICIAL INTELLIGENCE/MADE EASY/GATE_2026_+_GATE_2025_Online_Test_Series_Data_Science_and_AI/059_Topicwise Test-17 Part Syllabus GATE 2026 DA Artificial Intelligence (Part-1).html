Q12 MCQ Not checked
Technical • Data Science & Artificial Intelligence • +2 / -0.66
Not attempted

In Greedy Best-First Search, the node to be expanded next is selected based on:

Correct answer: C
Solution

(b)
Greedy Best-First Search is an informed search strategy that expands the node that appears to be closest to the goal, using a heuristic function.
• It uses a heuristic estimate h(n), which estimates the cost from node n to the goal.
• The algorithm selects the node with the lowest heuristic value h(n), assuming that lower h(n)
means closer to the goal.
