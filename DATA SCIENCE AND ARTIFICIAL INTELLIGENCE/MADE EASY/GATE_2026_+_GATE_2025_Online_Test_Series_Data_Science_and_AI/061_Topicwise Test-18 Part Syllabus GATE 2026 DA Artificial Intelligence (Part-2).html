Consider the following statements:
S1 : In variable elimination, factors are multiplied before summing out a variable.
S2 : Variable elimination gives the exact value of posterior probability for the query variable(s), given evidence.
Which of the following is correct?

Correct answer: C
Solution

(c)
• In variable elimination, we first multiply all factors involving the variable to be eliminated.
Only after this, we perform summation over that variable. This maintains the correctness of marginalization. So, S1 is correct.
• Variable elimination is an exact inference method. It computes the true posterior distribution of the query variable given the observed evidence, without using approximations. So, S2 is correct.

Which of the following algorithms is used in Exact Inference methods?

Correct answer: D
Solution

(d)
• Variational Inference approximates the true distribution by optimizing over a simpler family of distributions, making it a fast but approximate method.
• Gibbs Sampling is an approximate inference method that uses MCMC to draw samples from the posterior distribution. It is useful when exact inference is computationally infeasible.
• Loopy Belief Propagation applies belief propagation in graphs with cycles, but results are only approximate and not guaranteed to converge.
• Belief Propagation performs exact inference in tree-structured Bayesian networks by passing messages between nodes. It guarantees correct marginal probabilities when the graph has no loops.

S2 : The samples generated before reaching the stationary distribution are called burn-in samples.
Which of the following is correct?

Correct answer: C
Solution

(c)
• This is the Markov property, where the process is memoryless.
The next state depends solely on the current state, not on how it got there. So, S1 is correct.
• Burn-in samples are early outputs from the Markov chain before it reaches equilibrium.
These samples are discarded to ensure unbiased inference. So, S2 is correct.

Solution

Consider the following statements:
I. Gibbs sampling avoids computing the full joint distribution by sampling from the conditional distribution of each variable.
II. Gibbs sampling always converges quickly, even when the variables are highly correlated.
III. In Gibbs sampling, each variable is sampled conditionally on the current values of all other variables.
How many of the above statements are correct?

NAT
Correct answer: 2
Solution

2
• Gibbs sampling is a type of Markov Chain Monte Carlo (MCMC) method.
It avoids computing the full joint distribution by sampling each variable from its conditional
distribution, keeping all other variables fixed. So, Statement I is correct.
• Gibbs sampling does not always converge quickly.
If variables are highly correlated, the Markov chain may mix slowly, leading to slow
convergence. So, Statement II is incorrect.
• In Gibbs sampling, each variable is indeed sampled conditionally, based on the current values
of all other variables. This is the core mechanism of the algorithm. So, Statement III is correct.
